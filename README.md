## Vue2 Tutorial For IISI(CWB)

### Prerequisities
> nodejs & npm & webpack

### Global Installing 
``` npm install package.json```


### Running the tests

#### In example-html & example-vue1 
``` cd 'project folder' ```  

``` node web-app.js ```

#### In example-vue2 ~ example-vue6
``` cd 'project folder' ```  

``` npm install package.json```

``` webpack  --config webpack.config.js ```

``` npm test ```
